# GitHub Quick Setup — 5 Minutes

_repo is pushed. Now do these 5 steps on github.com:_

## Step 1: Set Description & Topics (1 min)

Go to: https://github.com/ParsaSmahmoudi/Hyperion

1. Click the ⚙️ gear icon next to "About"
2. Fill in:
   - **Description:** `9-stage reasoning amplification engine for LLMs — 5-30x deeper, more accurate responses`
   - **Website:** `https://github.com/ParsaSmahmoudi/Hyperion`
   - **Topics:** `llm` `reasoning` `ai` `agent` `chain-of-thought` `openai` `ollama` `openrouter` `async` `deep-learning`
3. Click "Save"

## Step 2: Create Release v2.0.0 (2 min)

Go to: https://github.com/ParsaSmahmoudi/Hyperion/releases/new

1. **Choose a tag:** `v2.0.0` (should already exist from our push)
2. **Release title:** `Hyperion v2.0.0 — The Reasoning Amplification Engine`
3. **Description:** Copy from `RELEASE_v2.0.0.md` in the repo
4. Check "This is a pre-release" (optional, for first release)
5. Click "Publish release"

## Step 3: Add Social Preview Image (1 min)

Go to: https://github.com/ParsaSmahmoudi/Hyperion/settings

1. Scroll to "Social preview"
2. Upload a 1280x640 image (or skip for now — GitHub auto-generates one from README)

## Step 4: Enable GitHub Pages (optional, 1 min)

Go to: https://github.com/ParsaSmahmoudi/Hyperion/settings/pages

1. Source: "Deploy from a branch"
2. Branch: `main`, folder: `/ (root)`
3. Click "Save"

## Step 5: Pin the Repo (optional, 30 sec)

Go to: https://github.com/ParsaSmahmoudi?tab=repositories

1. Click "Customize your pins"
2. Select "Hyperion"
3. Click "Apply"

## Done!

Your repo is now live at: **https://github.com/ParsaSmahmoudi/Hyperion**
